# Spin Anything

Spin text, images or anything else easily.

## Inputs

- img
- text
- size: default = '100'
- relativeSize
- speed: default = '3'
- color: default = '#000'
- weight: default = '300'
- fontSize: default = '20'
- relativeFontSize
- fontStyle: default = 'normal'

## Example

```html
<spinner
  text="Parameter"
  img="/assets/img/dragon.png"
  relativeSize="15"
  speed="5"
  color="blue"
  weight="bold"
  relativeFontSize="2"
></spinner>
```
